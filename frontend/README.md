# Libreville Eats Frontend

Front-end Next.js pour l'application Libreville Eats.

## Installation

```bash
cd frontend
npm install
```

## Lancement

```bash
npm run dev
```

## Variables d'environnement

Ajoutez un fichier `.env.local` dans `frontend/` avec :

```
NEXT_PUBLIC_API_URL=http://localhost:5000/api
```
